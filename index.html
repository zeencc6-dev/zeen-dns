📱 ডিভাইস অনুযায়ী পরিপূর্ণ সেটআপ গাইডলাইন:

  1. অ্যান্ড্রয়েড (Android): ফোনের Settings > Network & internet > Private DNS অপশনে যান। সেখান থেকে Private DNS provider hostname সিলেক্ট করুন এবং ওপর থেকে কপি করা এন্ডপয়েন্টটি পেস্ট করে Save করুন।
  2. আইফোন / আইপ্যাড (iOS): ফোনের Settings > Wi-Fi এ গিয়ে আপনার কানেক্টেড ওয়াইফাই নেটওয়ার্কের পাশের 'i' আইকনে ট্যাপ করুন। এরপর Configure DNS অপশনে গিয়ে এটি Manual করুন এবং জেনারেট হওয়া হোস্টনেমটি যুক্ত করুন।
  3. উইন্ডোজ পিসি (Windows 10/11): কম্পিউটারের Settings > Network & Internet > Wi-Fi/Ethernet > Hardware properties এ যান। এরপর DNS server assignment এ Edit ক্লিক করে Manual সিলেক্ট করুন এবং Preferred DNS-এ হোস্টনেম বসিয়ে দিন।
  4. ম্যাকবুক (Mac): System Settings > Network > Wi-Fi > Details > DNS ট্যাবে গিয়ে '+' আইকনে ক্লিক করে আপনার কাস্টম ডিএনএস এন্ডপয়েন্ট বা হোস্টনেমটি যোগ করুন।
